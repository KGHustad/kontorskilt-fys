### Kommandoer
For å kjøre pdflatex
```
make
```

For å vise PDF-filen (og generere den hvis nødvendig)
```
make show
```

For å slette unødvendige filer generert av LaTeX
```
make clean
```

For å slette alle filer som ikke kan genereres på nytt
```
make clean-all
```
